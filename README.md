## installation / usage
✦ . 　⁺ 　 . ✦ . 　⁺ 　 . ✦

to install the necessary dependencies, run: `npm install -g gorthaur` or simply `npm i gorthaur`.

after that, you can use the CLI by executing `gorthaur` in your terminal.
